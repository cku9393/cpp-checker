# raw_engine_v1 package

Standalone first-working-version raw primitive engine package.

Included:
- `raw_engine_v1.cpp`: helpers, validators, raw primitives, planner v1, microcases, stress tests, small fuzz
- `build_and_run.sh`: compile and run self-tests

Build:
```bash
bash build_and_run.sh
```

Output on success:
```
raw_engine_v1 microcases+stress passed
```
